Juan's Movie API

The web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

Endpoints Table

Request URL Response
GET /movies Get all movies
GET /movies/:id Get specific movie
POST /movies Create a new movie.
Request Body Example:
{
  "title": "Inception",
  "director": "Christopher Nolan",
  "genre": "Science Fiction",
  "year": 2010,
  "description": "A thief who steals corporate secrets through the use of dream-sharing technology..."
}
PUT /movies/:id Update movie.
Request Body Example:
{
  "title": "Inception Updated",
  "director": "Christopher Nolan",
  "genre": "Science Fiction",
  "year": 2010,
  "description": "A thief who steals corporate secrets through the use of dream-sharing technology..."
}
DELETE /movies/:id Delete a movie
GET /genres Get all genres
GET /genres/:name Get specific genre
POST /genres Create a new genre
PUT /genres/:name Update genre
DELETE /genres/:name Delete a genre
GET /directors Get all directors
GET /directors/:name Get specific director
POST /directors Create a new director
PUT /directors/:name Update director
DELETE /directors/:name Delete a director
GET /users Get all users
GET /users/:id Get specific user
POST /users Create a new user.
Request Body Example:
{
  "username": "johndoe",
  "password": "password123",
  "email": "johndoe@example.com",
  "birthday": "1990-01-01"
}
PUT /users/:id Update user.
Request Body Example:
{
  "username": "john_doe_updated",
  "password": "newpassword123",
  "email": "john_doe_updated@example.com",
  "birthday": "1990-01-01"
}
DELETE /users/:id Delete a user
POST /users/:id/favorites Add a movie to user's favorites.
Request Body Example:
{
  "movieId": "60c72b2f4f1a4d3c4c8e5f3b"
}
DELETE /users/:id/favorites/:movieId Remove a movie from user's favorites
